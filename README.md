## Keybindings
| Keys | Action |
| :--- | :--- |
| <kbd>Super</kbd> + <kbd>T</kbd> | Launch terminal emulator (Alacritty) |
| <kbd>Super</kbd> + <kbd>E</kbd> | Launch file manager (Dolphin) |
| <kbd>Super</kbd> + <kbd>L</kbd> | Lock Screen (Hyprlock) |
